Inlämning 1b

    • Element är hur vi skriver kod i HTML. Ett element börjar med en öppningstagg
      och avslutas med >, inom dessa taggar skriver vi in själva koden.
    • I varje element finns ett attribut. Attribut bestämmer vad elementet
      skall göra.
    • Ibland kan man ge en attribut ett värde. Detta kan man göra för att
      länka till en bild eller skapa en länk.
  2. HTTP står för Hypertext Transfer Protocol och är ett kommunikationsprotokoll.
    HTTP bestämmer hur meddelanden utformas och skickas men också vad för svar som ska ges.
    Exempelvis när man skrivit in en url för att komma till en hemsida så är det HTTP
    som säger hur hemsidan ska hämtas.
  3. W3C är ett internationellt samarbete mellan W3C:s medlemsorganisationer, regeringar
    standardiseringsorgan m.fl. som arbetar med att utveckla standards, tekniska protokoll samt
    programvara till webben för att kunna använda den till sin fulla potential.
  4. www står för world wide web och är tjänsten som gör webbsidor och datafiler tillgängliga
    från Internet eller Intranät.
    • URL
    • Uniform Resource Locator, detta är en hemsidas unika sökväg.
    • A. Berättar vilket protokoll som används
      B. Detta är domännamnet.
      C. Vilken Resurs som används på hemsidan.
  6. Vi använder det främst till att ladda upp hemsidor till webbplatser.
  7. HTTP använder sig av request and respons modellen som i kort kan förklaras som att en användare
    skickar en förfrågan till ex. en server, att få se en hemsida. Servern svarar då på förfrågan genom att
    skicka hemsidans information så att användaren kan se hemsidan.
  8. HTML står för HyperText Markup Language och är också det språk som används oftast när man skriver
    hemsidor. HTML kallas ibland också för ett sidbeskrivningsspråk.
